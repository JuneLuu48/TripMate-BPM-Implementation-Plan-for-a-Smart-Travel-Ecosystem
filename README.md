# Smart Travel Planning System

## This project focuses on designing and modelling a smart travel planning platform that integrates multiple travel services into a single application. The system aims to simplify trip organisation by allowing users to plan, book, and manage travel activities through a unified interface.

## The project uses Business Process Modelling (BPMN) to analyse and redesign travel planning workflows. It presents both AS-IS and TO-BE process models to illustrate how current travel planning processes can be improved through automation and system integration.

### The platform enables users to:

Create and manage trips within the application

Search and compare flight options based on price and reviews

Book accommodation and restaurants

Discover places to visit and activities

Process payments and receive booking confirmations

Manage travel budgets and preferences

Receive AI-powered travel suggestions based on user activity

Integrate with real-time transport applications for navigation and journey planning

The redesigned TO-BE process improves efficiency by reducing manual steps, integrating services into a single platform, and providing users with personalised recommendations.

### Key Features

Integrated flight, accommodation, and activity booking

BPMN-based process modelling for system optimisation

Payment processing and booking confirmation workflows

AI-assisted travel recommendations

Trip budget management and user preference tracking

Real-time transport integration for navigation

### Tools & Methods

BPMN process modelling

Business process analysis (AS-IS / TO-BE modelling)

System workflow design

Digital service platform design

### Goal

The goal of this project is to demonstrate how process modelling and system integration can improve the user experience in travel planning while creating a more efficient and scalable digital travel service platform.
